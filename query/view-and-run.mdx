---
title: 'View and Run a Query'
description: 'How to view existing queries and run them in your spreadsheet'
---

To execute a saved query, please follow these steps:

1. Launch the Viasheets Extension in Google Sheets.
2. Choose the desired Organization for which you intend to run a query.
<Frame>
  <img src="/images/sheets/screen-1.png" style={{ borderRadius: '0.5rem' }} />
</Frame>
3. Click the Navbar button and choose "Existing Queries."
<Frame>
  <img src="/images/sheets/screen-2.png" style={{ borderRadius: '0.5rem' }} />
</Frame>
4. Allow time for the queries to load.
5. Select the query you wish to run and click "Run." This action will confirm the destination cell for data retrieval and trigger the query execution.
<Frame>
  <img src="/images/sheets/screen-10.png" style={{ borderRadius: '0.5rem' }} />
</Frame>

## Deleting a Query

<Note>
If you want to delete a query, click "Delete".
</Note>
